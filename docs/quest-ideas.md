# Головоломка

Маг настраивает волшебное зеркало на нужную частоту, чтобы отправить приключенца в нужное место

Как это можно обыграть?

1. Игроку нужно поймать нужную магическую частоту, которая расшифрует сообщение, которое хочет передать персонаж герою

2. После этого нужно настроить магические кристаллы и выполнить телепортацию

3. Если сделать что-то не так – персонаж телепортируется в рандомное место (смешная анимация смерти)
